# Academic Scheduler

I recently thought of an effort to organize and record future tasks and events that's available to the public. Therefore, anyone can view them through the website and reduce the chance of missing out on them. Moreover, be aware of any new changes before anyone else, and discuss about it instead of complaining at the last minute.

# This is how it's gonna work...

Anyone can add a schedule using G-form, but only selected volunteers (preferably CRs) can edit the Excel sheet. Their work will be to delete the completed schedules. I have attached the Excel sheet view as Schedule, the Google Form view for adding new tasks to the Schedule, and the Website view that links to both the sheet and form. Join me at the Telegram Channel https://t.me/oUITS_ORnU for the edit access.

# Conclusion

This is purely a crowdsourcing project and therefore, highly dependent on the people using it. Only those willing to utilize it will benefit from it. It is just the first demo of the idea therefore Suggestions and Criticisms are highly appreciated.
This is also intended to be an open-source project, so I appreciate anyone willing to contribute in any way they want to. For example, you are against the idea of me having admin access over all the Forms and Sheets, you can provide me the link of your admin access, I'll add them over mine. It doesn't really matter.
